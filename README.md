# tic-tac-toe
Simple Console Application with 2 different modes.

1. Singleplayer (VS Computer)
2. Multiplayer 

<h4>RUN<h4>
  Use the following command in command line to run the Program
  

```
g++ main.cpp def.cpp; ./a
```
  <p> or you may use </p>
  
```
if ($?) { g++ main.cpp def.cpp -o main } ; if ($?) { .\main }
```
